AirDrawer – Project Description

AirDrawer is an AI-powered web application that allows users to draw in the air using hand gestures captured through a webcam. The system uses computer vision and hand-tracking technology to detect finger movements and convert them into digital drawings in real time.

This project leverages advanced hand gesture recognition to create a touchless drawing experience, making it useful for creative design, education, presentations, and interactive applications.

The application is built using modern web technologies such as HTML, CSS, JavaScript, and AI-based hand tracking libraries. It detects hand landmarks from live video input and tracks finger positions to simulate drawing on a virtual canvas.

Key Features
✋ Air Drawing with Hand Gestures
Draw on the screen by moving your finger in the air.
🎥 Real-Time Hand Tracking
Uses webcam input to detect hand movements instantly.
🎨 Virtual Canvas
Converts finger motion into digital strokes.
⚡ Touchless Interaction
No mouse or touch required—fully gesture-based.
🎯 AI-Based Recognition
Uses hand detection models for accurate tracking.
Technologies Used
HTML – Structure of the web application
CSS / Tailwind CSS – Styling and UI design
JavaScript – Core functionality
MediaPipe Hands – Hand tracking and gesture detection
Webcam API – Capturing real-time video input
Working Principle
The webcam captures live video input.
The hand-tracking model detects hand landmarks.
Finger movement is tracked continuously.
The system converts finger motion into drawing strokes.
The strokes appear on the virtual canvas in real time.
Applications
Digital drawing and design
Virtual whiteboards
Educational demonstrations
Gesture-based interaction systems
Contactless user interfaces
